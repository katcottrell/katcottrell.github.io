What is MATLAB?

"MATLAB® is a programming platform designed specifically for engineers and scientists. The heart of MATLAB is the MATLAB language, a matrix-based language allowing the most natural expression of computational mathematics."

—MathWorks official website

MATLAB, short for matrix laboratory, was developed by MathWorks founders in the 1970s and 1980s with the goal of creating a programming language for applied mathematics and matrix calculations.

MATLAB is great for:

  • Mathematical computation
  • Data visualization and graphics
  • Algorithm implementation
  • Computations involving matrix or vector formulations
  • Model-based design
  • Simulation
  • Data analysis
  • Science, engineering, and other industry applications
  • App and GUI development

MATLAB comes with a comprehensive library of mathematical functions and toolboxes for specialized applications like deep learning, image processing, aerospace, parallel computing, finance, signal processing, and many more.

How do I know MATLAB?

I learned MATLAB in my Computational Methods in Engineering course at Duke University. In the course, we learned to model physical systems, optimize parameters, perform numerical integration, find roots of equations, perform simultaneous equation solving, use finite difference methods, do matrix analysis, implement linear and dynamic programming, and find heuristic solutions.
