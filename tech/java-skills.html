Java was developed in the 1990s, originally with the goal of creating digital interactive television, but is now one of the most commonly used programming languages in the world. Java is great for:

  • Single-computer applications
  • Webpage applets
  • Fast execution
  • Readable, easy-to-learn code
  • Scientific and mathematical projects
  • Corporate projects
  • Projects benefitted by object-oriented programming

I learned Java in my Algorithms and Data Structures class at Duke, taught by Dr. Owen Astrachan. The course was taught with the goal of learning to design, develop, debug, and test Java programs, implement data structures, and evaluate the time and space complexity of iterative and recursively-defined algorithms.
