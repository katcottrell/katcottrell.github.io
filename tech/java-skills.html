What is Java?

"Java is the #1 programming language and development platform."

Java is an object-oriented programming language. It was created by James Gosling at Sun Microsystems in the 1990s, originally with the goal of creating digital interactive television. Today, Java is one of the most commonly used programming languages in the world.

Java is great for:

  • Single-computer applications
  • Webpage applets
  • Fast execution
  • Readable, easy-to-learn code
  • Scientific and mathematical projects
  • Corporate projects
  • Projects benefitted by object-oriented programming

How do I know Java?

I first learned Java in my Algorithms and Data Structures class at Duke. The course was taught with the goal of learning to design, develop, debug, and test Java programs; implement data structures; and evaluate the time and space complexity of iterative and recursively-defined algorithms.

I later used Java in my Software Design and Implementation course at Duke, in which I used Java to write many programs, with the goal of strengthening my software design skills.
