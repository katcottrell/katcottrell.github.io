What is Python?

"Python is an interpreted, object-oriented, high-level programming language with dynamic semantics."

—Python official website

Python is a general-purpose programming language general-purpose programming language. It was created by Guido van Rossum at Centrum Wiskunde & Informatica in the 1990s, and has quickly become one of the most popular programming languages in the world.

Python is great for:

  • Beginners to coding (but not only beginners!)
  • Backend web development
  • Data analytics
  • Mathematical and scientific projects
  • Projects that use one of Python’s many libraries, including packages for data analytics, image processing, machine learning, and more!

How do I know Python?

I originally taught myself the basics of Python when I learned that an engineering course taught in MATLAB I had taken replaced the introductory computer science course taught in Python at my university. I didn’t want to be missing Python skills, so I self-taught the basics using Eric Matthes’s Python Crash Course. Later, I learned a great deal more about Python in my computation of linear algebra class, where I used NumPy, and my Everything Data class, where I learned Pandas, machine learning methods, and much more using the Python language.
