My Database Skills

What are databases?

“A database is an organized collection of structured information, or data, typically stored electronically in a computer system. A database is usually controlled by a database management system (DBMS).”

— Oracle official website

Databases are a great way for businesses and organizations to store massive datasets with minimal redundancy and lightning speed access. To maintain and access the data in a database, users have a wide array of tools to choose from. In 2022, the top ten most popular database management systems (DBMSs) were Oracle, MySQL, Microsoft SQL Server, PostgreSQL, MongoDB, Redis, IBM Db2, Elasticsearch, Microsoft Access, and SQLite ( source).

My experience with databases

MySQL

I learned MySQL in my Introduction to Database Systems course at Duke. MySQL, short for "Structured Query Language," is one of the most common DBMSs in widespread use.


Microsoft SQL Server

I learned MS SQL Server during my internship at Nextech Systems. MSSQL Server is an intuitive and classic solution for database management.


MongoDB

I learned MongoDB in my Introduction to Database Systems course. MongoDB has comprehensive documentation and is an accessible option for many businesses.


Microsoft Access

I learned Microsoft Access during my internship at Short Block Technologies. The basic version of Microsoft Access is free with Microsoft 365, and it’s great for managing any large dataset that a company or organization may need to store and access.


XML

Although XML is not a DBMS, it is another great tool for managing data that I learned in my Introduction to Database Systems course. XML, for “eXtensible Markup Language,” is a markup language for data storage that’s great for tree-shaped data.
