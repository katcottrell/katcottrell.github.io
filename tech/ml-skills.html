I can easily, however, explain my history with machine learning (ML). I first began learning about machine learning when I took on a position with the Duke Undergraduate Machine Learning executive board. As an executive board member, I planned and hosted events to educate undergraduates on machine learning and big data topics. One of these events was our annual Duke Machine Learning Day, a day dedicated to Machine Learning, attended by undergraduate students, graduate students, faculty, and national experts in machine learning alike. Others included bimonthly lectures by machine learning researchers, all of which I promoted and attended. I also attended events hosted by one of our sponsors, InfiniaML, which taught ML basics and allowed me to speak with both new and experienced developers about working in ML.

In the fall of 2019, I took Introduction to Machine Learning at Duke University, taught by Dr. Lawrence Carin, which included both a series of lecture modules and through in-person learning experiences. In this course I was able to learn a great deal more about the mathematics and software behind machine learning, as well as the applications of such technology.

For more information on machine learning, I encourage you to look into Dr. Carin’s online Coursera course on machine learning. If you are interested in my personal history with machine learning, please keep checking back—I hope to post more information on my experiences and work with ML soon!
